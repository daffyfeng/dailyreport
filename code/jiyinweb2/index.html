关于公司

基因资本是由好未来（学而思）联合创始人曹允东2014年创立的风险投资基金，管理团队由成功企业家和资深投资人组成，旨在寻找优秀基因，协助教育行业和TMT领域的创业公司成就伟大事业。基因资本乐于帮助那些对创业怀揣热情、对行业发展趋势有深刻理解和敏锐意识、对基因资本投资理念认可的创业伙伴实现他们的梦想。
我们的投资理念是关注创业基因：优秀的创业团队具有优秀的创业基因，优秀的创业基因是企业长期发展的核心因素。
我们的使命是用资本成就未来：把最好的资源提供给最优秀的创业基因，为他们保驾护航，陪伴他们成长，帮助他们成功。

我们的团队

曹允东

管理合伙人

在创办基因资本前，曹先生曾创办学而思教育集团并担任董事和总裁，并在2010年10月带领学而思成功在纽交所上市，目前市值约为230亿美元。2011年6月，曹先生创办珍品网，2014年12月创办乐学高考，专注教育培训。曹允东先生毕业于北京大学。
